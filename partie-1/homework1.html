1 Présentation


Les objectifs de ce cours/TP sont d vous montrer comment installer et configurer un serveur web pour qu'il puisse interpréter dse scripts PHP. Nous verrons également comment fonctionne le language PHP et comment utiliser PHP conjointement avec AJAX. Enfin, dans la dernière partie du TP nous expérimenterons d'autres technologies concurrentes à PHP tel que l'ASP.Net de Microsoft.

1.1 Pré-requis


Pour réaliser ce cours/TP, vous avez besoin d'un serveur web fonctionnel. Vousutiliserez le serveur http sous Linux que vous avez configuré lors du cours/TP 1.
Vous aurez également besoin d'un éditeur de texte pour écrire le code ded vos pages. Vous pouvez par exemple utiliser un des logiciels Notepad++, ConText, Quanta+, WebExpert, SciTE,...
Pour réaliser ce cours/TP, vus garderez les groupes constitués lors du premier cours/TP.

2 Installation et configuration du server HTTP

Pour nos excercices, il nous faudra disposer d'un server http acceptant les spirts PHP et donnant accès à un base de données. Nous installerons donc PHP et mysql dans notre machine virtuelle linux-debian. Si vous avez installé le forum phpBB lors du cours/TP1, vous avez déjà fait une partie de ce travail.

2.1 Installation

Commençons par ajouter mysql à notre installation:

>apt-get install mysql-server
ou
>aptitude install mysql-server

Nous devrons également installer les modules php pour apache. Pour cela, vous devez installer une interpréteur php à l'aide de la commande suivante:

>apt-get install php5
ou
>aptitude install php5
